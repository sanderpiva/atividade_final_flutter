# 🚗 Radar do Combustível: Inteligência na Hora de Abastecer

O **Radar do Combustível** é um aplicativo mobile desenvolvido em **Flutter** que auxilia motoristas a tomarem a decisão financeira mais inteligente no posto de gasolina. O app calcula a relação de custo-benefício entre Etanol e Gasolina de forma personalizada para cada veículo.

## 🎯 Proposta do Projeto
O objetivo é fornecer uma ferramenta ágil e útil para o dia a dia, capaz de:
* **Calcular a Vantagem Econômica ($):** Identificar qual combustível é mais barato por quilômetro rodado com base no veículo selecionado.
* **Transparência de Dados:** Exibir o cálculo detalhado diretamente na tela para o usuário.
* **Compartilhamento Social:** Permitir o envio dos resultados e do nome do posto via WhatsApp ou outras redes sociais.

---

## ✨ Funcionalidades e UX
A aplicação foi projetada para ser intuitiva e de fácil manipulação:
* **Seleção de Veículo:** Ajuste do cálculo conforme o consumo específico de diferentes modelos.
* **Modo Compartilhamento:** Ao selecionar o *checkbox* antes do cálculo, o app abre automaticamente as opções de compartilhamento do sistema.
* **Interface Dinâmica:** Feedback visual imediato sobre a melhor escolha de abastecimento.

---

## 🛠️ Estruturas e Tecnologias
Para a construção desta solução, foram empregados componentes fundamentais do ecossistema Flutter:
* **Framework:** Flutter (Dart).
* **Entrada de Dados:** `TextFormField` para inserção de preços e nomes de postos.
* **Seleção:** `DropdownButtonFormField` para escolha da frota/veículo.
* **Lógica de Fluxo:** `Checkbox` para controle da funcionalidade de compartilhamento.

### 💡 Considerações
Este projeto é um exemplo prático de como a **observação de dados** (neste caso, preços dinâmicos e consumo veicular) pode ser transformada em uma ferramenta de apoio à decisão de negócio (ou pessoal). O app atua como a lente que processa variáveis externas para gerar um insight financeiro imediato.
